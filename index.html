CAPABILITIES

Component Enumeration

Direct hardware enumeration APIs bypass the OS abstraction entirely. CPU topology, PCI hierarchy, memory channels, firmware strings at raw device level. Serial cross-reference detects cloning and tampering instantly.

Integrity Verification

SHA-256 manifest cryptographically binding the audit to the exact hardware state at execution time. Run it twice on identical hardware: identical output, byte for byte.

RAM Telemetry

Deep allocation and state memory scanning. Full memory channel enumeration, speed, timing, and serial identification at the hardware level.

Security Notice — Defender Alert

KALIX dynamically extracts a signed Windows driver and interfaces firmly with the kernel level, which may occasionally trigger a Windows Defender SmartScreen warning correctly prioritizing hardware safety. This behavior is standard for zero-dependency kernel audit tools. It carries zero telemetry and requires no installation.
